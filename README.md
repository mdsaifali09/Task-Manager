📌 Task Manager App (MERN Stack)

A full-stack Task Manager Application built using React (Frontend), Node.js + Express (Backend), and MongoDB (Database).
The app allows users to create, view, update, and delete tasks efficiently.

🚀 Live Demo
Frontend: https://task-manager1-9i6a.vercel.app/
Backend API: https://task-manager-500v.onrender.com

⚡ How to Run Locally
1️⃣ Clone repo
git clone https://github.com/mdsaifali09/task-manager.git

2️⃣ Backend setup
cd backend
npm install
npm start

Create .env file:

MONGO_URL=your_mongodb_connection_string
PORT=5000

3️⃣ Frontend setup
cd frontend
npm install
npm run dev
